# Portfolio Website
<u>Link to deployed website</u>: https://cn-guy.github.io/

This repo contains all of the code for my portfolio website.

The website is primarily made up of HTML and raw CSS, with some JavaScript implemented to provide interactivity for some features.

<u>__To run the code locally__</u>:
- Download and open the repo with VS Code
- Download the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension by Ritwick Dey
- On the bottom right, click the "Go Live" button. It will open the code in your browser locally
